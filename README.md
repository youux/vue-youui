# 组件库
## 创建模板
```
yarn plop
```

### 创建storybook
```
npx -p @storybook/cli sb init
yarn add vue-loader vue-template-compiler -D -W
```

### 单元测试
安装：
```
yarn add jest @vue/test-utils vue-jest babel-core@bridge babel-jest -D -W
```


### lerna使用
安装：
```
yarn global add lerna
```

初始化
```
lerna init
```
